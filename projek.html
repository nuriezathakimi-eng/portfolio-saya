Project - User AWB (Air Waybill) - (Local)

Ini adalah projek kajian saya berkaitan dengan kerja saya sebagai seorang pembantu gudang.
Projek ini tidak boleh diakses secara online kerana ia adalah projek tempatan (local).
Saya telah menggunakan beberapa teknologi seperti HTML, CSS, dan PHP dalam pembangunan projek ini.
Projek ini belum ada Database lagi kerana saya masih dalam proses pembelajaran.
Projek ini bertujuan untuk memudahkan pengurusan dan penjejakan AWB di gudang saya.

AWB ni straight forward, user masuk laman web (link) projek.
User akan diberi paparan beberapa input untuk di masukkan.
User di sini adalah pengguna sistem AWB yang terdiri daripada staf pembantu gudang.
Pembantu Gudang akan mengisi Maklumat Penerima, Jumlah Kotak, Tarikh, Nombor Invoice
dan Catatan jika ada.

Dengan bantuan teknologi PHP, sistem ini dapat menggunakan HTML & CSS untuk
mengeluarkan hasil dalam bentuk format file .pdf
Sistem ini sesuai untuk syarikat logistik yang mempunyai kekerapan membuat penghantaran bungkusan yang banyak.
Pembantu gudang boleh menggunakan sistem ini untuk mencetak AWB dengan cepat dan mudah.
Pembantu gudang hanya perlu mengisi maklumat yang diperlukan dan sistem akan menjana AWB dalam format PDF
mereka tidak perlu tulis AWB secara manual & tulis perkara yang sama berulang kali.
Dengan adanya sistem ini, ia dapat membantu mempercepatkan proses penghantaran bungkusan.


Antara bantuan yang saya gunakan adalah AI kerana ia membantu saya dalam menganalisis data dan membuat keputusan yang lebih baik.
Projek ini adalah langkah awal saya dalam dunia pembangunan web dan saya berharap dapat terus belajar dan berkembang dalam bidang ini.
Saya berharap dapat terus memperbaiki kemahiran saya dalam pembangunan web dan menambah baik projek ini dari semasa ke semasa.
